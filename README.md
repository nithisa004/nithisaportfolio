# Portfolio - Nithisa Devi J

Welcome to the personal portfolio of **Nithisa Devi J**, a Full-Stack Developer & Cybersecurity Enthusiast.

## About
This is a modern, responsive web portfolio built with HTML, CSS, and JavaScript. It features a custom cyber-premium dark theme, glassmorphism UI elements, and a dynamic hero design.

## Features
*   **Custom Design System**: Pitch black cyber premium theme with neon accents.
*   **Responsive Layout**: Fully responsive across all devices.
*   **Interactive UI**: Micro-animations, hover effects, and a simulated mock IDE.
*   **Projects Showcase**: Dedicated sections to highlight Full-Stack Development and Cybersecurity projects.

## Technologies Used
*   HTML5
*   CSS3 (Vanilla)
*   JavaScript
*   Font Awesome Icons
*   Google Fonts (Space Grotesk, Fira Code, Playfair Display)

## Setup & Deployment
To view the portfolio locally:
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
